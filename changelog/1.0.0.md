# Version 1.0.0 release

Commit range: `3069a66..0e1c086`

## Highlights
- Refactored core architecture and naming conventions for clarity and consistency across compiler, parser, and process modules.
- Fixed three long-standing rendering bugs: display math mangling in embedded markdown, datetime formatting, and backslash/footnote processing.
- Added developer-facing and user-facing documentation.
- Improved table styling in rendered output.
